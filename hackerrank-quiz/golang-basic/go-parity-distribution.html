2. Go: Parity Distribution

Implement a function Server which runs a server with the following characteristics:

  • It accepts values through a channel. Each value is an int32. You should describe a type 'in' and initialize a global variable serverChan
  • Depending upon whether the value is odd or even, it directs them to oddChan and evenChan.

Example

arr = [1,2,3,4,5]

oddChan receives [1,3,5] and evenChan receives [2,4]. The values are printed to STDOUT.

Function Description

Complete the function Server in the editor below. The function is void.

Server has no parameters.

Constraints

  • 1 ≤ length of arr ≤ 1000
  • 1 ≤ arr[i] ≤ 107

Input Format For Custom Testing

The first line contains an integer, n, the number of elements in arr.
Each line i of the n subsequent lines (where 0 ≤ i < n) contains an int32, arr[i].
